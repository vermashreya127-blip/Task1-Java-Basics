Java Basics & OOP Fundamentals

Project Overview

This project was developed as part of the Alfido Tech Java Developer Internship. It demonstrates the basic concepts of Java programming and Object-Oriented Programming (OOP) through practical examples.

Programs Included

Main.java

Prints "Hello, World!" to verify Java installation and execution.

LoopExample.java

Demonstrates the use of loops in Java.

ConditionExample.java

Demonstrates conditional statements (if-else).

ArrayExample.java

Demonstrates array declaration, initialization, and traversal.

Student.java

Demonstrates classes and objects in Java.

AnimalTest.java

Demonstrates inheritance and polymorphism.

EncapsulationExample.java

Demonstrates encapsulation using private variables with getter and setter methods.

OOP Concepts Covered

- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation

Technologies Used

- Java
- JDK 21
- Visual Studio Code (VS Code)
- GitHub

How to Run

Compile a Java file:

javac FileName.java

Run the program:

java FileName

Example:

javac Main.java
java Main

Learning Outcomes

- Understanding Java syntax and program structure
- Working with loops and conditional statements
- Using arrays in Java
- Creating classes and objects
- Implementing inheritance and polymorphism
- Applying encapsulation using getters and setters
